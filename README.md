# 🎓 Student Attendance Management System

A Python-based **Student Attendance Management System** designed to automate student record management, attendance tracking, report generation, and data persistence using SQL database integration.

This project helps reduce manual attendance work by providing an efficient system for managing student information and attendance records.

---

## 🚀 Features

### 🔐 Admin Login
- Secure admin authentication system
- Controlled access to management modules

### 👨‍🎓 Student Management
- Add new students
- View student records
- Update student details
- Delete student records

### 📅 Attendance Management
- Mark daily attendance
- Store attendance records with dates
- View date-wise attendance
- Calculate attendance percentage

### 📊 Report Generation
- Daily attendance reports
- Student-wise attendance reports
- Overall attendance summary

### 📁 File Handling
- Export attendance records into CSV files
- Maintain attendance backup reports

### 🗄️ Database Management
- SQLite database integration
- Efficient CRUD operations
- Structured data storage
